# Space 2048 IT Project

**Space 2048** - a puzzle game built in **Unity**.  
This project was created as part of a game programming course, aiming to combine the classic **2048 mechanics** with a unique space-themed visual style.

---

## About the Game

Players slide tiles on a board, merging them into higher values until reaching the 2048 goal.  
**Space 2048** stands out with:

- Cosmic 2D graphics  
- Smooth animations when tiles merge  
- Sound effects for moves
- Intuitive UI and simple controls

---

## Features

- Classic 2048 gameplay mechanics  
- Score counter 
- Visual and audio feedback for each move  
- Space-themed design and graphics  
- Easy to expand with new features (e.g., power-ups, special animations)

---

## Technologies

- **Engine:** Unity  
- **Language:** C#  
- **Graphics:** custom 2D assets  
- **Audio:** interactive sound effects
